# PRD: 개인 자산관리 대시보드

## 1. 개요

개인이 보유한 다양한 자산(주식, 예금, 부동산, 암호화폐)을 한눈에 파악하고 관리할 수 있는 웹 대시보드.
Google Sheets를 데이터 소스로 활용하여 별도 DB 없이 간편하게 자산 데이터를 관리한다.

## 2. 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Next.js 14 (App Router) |
| 언어 | TypeScript |
| 스타일링 | Tailwind CSS |
| 차트 | Recharts |
| 데이터 소스 | Google Sheets API |
| 배포 | Vercel (예정) |

## 3. 자산 카테고리

| 카테고리 | 관리 항목 |
|----------|-----------|
| 주식 | 종목명, 수량, 매수가, 현재가, 수익률 |
| 예금 | 은행명, 상품명, 원금, 이자율, 만기일 |
| 부동산 | 물건명, 매입가, 현재 시세, 대출 잔액 |
| 암호화폐 | 코인명, 수량, 매수가, 현재가, 수익률 |

## 4. 주요 기능

### 4.1 대시보드 (메인)
- 총 자산 금액 표시
- 자산 카테고리별 비중 (도넛 차트)
- 월별 자산 변동 추이 (라인 차트)
- 카테고리별 요약 카드

### 4.2 자산 현황
- 카테고리별 상세 테이블
- 각 자산의 현재가, 평가금액, 수익률 표시
- 수익/손실 색상 구분 (양: 초록, 음: 빨강)

### 4.3 수익률 차트
- 전체 포트폴리오 수익률 추이
- 카테고리별 수익률 비교 바 차트
- 기간 필터 (1개월, 3개월, 6개월, 1년, 전체)

### 4.4 포트폴리오 비중
- 카테고리별 비중 (도넛/파이 차트)
- 개별 자산별 비중 (트리맵 또는 테이블)
- 목표 비중 vs 현재 비중 비교

## 5. 데이터 연동 (Google Sheets)

### 시트 구조 (예시)
- **Sheet 1: 주식** — 종목명 | 수량 | 매수단가 | 현재가 | 메모
- **Sheet 2: 예금** — 은행 | 상품명 | 원금 | 이자율(%) | 만기일
- **Sheet 3: 부동산** — 물건명 | 매입가 | 현재시세 | 대출잔액 | 메모
- **Sheet 4: 암호화폐** — 코인명 | 수량 | 매수단가 | 현재가 | 메모
- **Sheet 5: 월별기록** — 날짜 | 총자산 | 주식 | 예금 | 부동산 | 암호화폐

### 연동 방식
- Google Sheets API v4를 통해 데이터 읽기
- Next.js API Route에서 서버사이드로 호출
- 클라이언트에서 새로고침 버튼으로 최신 데이터 반영
- 캐싱: 5분 간격 revalidate

## 6. 페이지 구조

```
/                  → 대시보드 (메인)
/stocks            → 주식 상세
/deposits          → 예금 상세
/real-estate       → 부동산 상세
/crypto            → 암호화폐 상세
/portfolio         → 포트폴리오 비중 분석
```

## 7. UI/UX 요구사항

- 반응형 디자인 (데스크탑 + 모바일)
- 다크 모드 기본
- 금액 단위: 원(₩) 기본, 천원/만원/억원 단위 자동 변환
- 로딩 상태 스켈레톤 UI
- 데이터 새로고침 버튼

## 8. MVP 범위 (1차)

1. Google Sheets 연동 및 데이터 파싱
2. 메인 대시보드 (총자산, 카테고리 비중 차트, 요약 카드)
3. 카테고리별 상세 테이블
4. 기본 수익률 차트

## 9. 향후 확장 (2차+)

- 실시간 시세 API 연동 (주식/암호화폐)
- 목표 자산 배분 설정 및 리밸런싱 알림
- 배당금/이자 수익 추적
- PDF 리포트 내보내기
- 환율 변환 (해외 자산)
